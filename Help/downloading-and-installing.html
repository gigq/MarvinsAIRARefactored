Downloading and Installing

How to install MAIRA Refactored (2.0) and keep it up to date.

Overview

This page explains how to download and install Marvin’s Awesome iRacing App – MAIRA Refactored (2.0). The process is similar to the original MAIRA app, but the download location is in a different place on GitHub.

Legal Stuff

This app has been downloaded and used by thousands of users without any issues, and a lot of effort has been spent in making sure it works.

This app controls the force feedback of your wheelbase. If you have a very strong wheelbase, this app can and will cause injuries if you are not careful.

This app does not do any automatic modifications to iRacing, but may require you to manually modify iRacing including disabling Force Feedback in iRacing, and possibly modifying your iRacing app.ini file if you have a Logitech wheel.

The app may require you to update your wheelbase firmware and/or software to work. This app does sometimes require additional components to be installed directly from Microsoft, depending on your system. These modifications and updates may cause iRacing or your wheel or your computer to stop functioning as expected.

Step 1 — Download the Installer

You can always get the latest version of MAIRA Refactored from the GitHub Releases page:

https://github.com/mherbold/MarvinsAIRARefactored/releases/latest

On that page, look for the Assets section and download the installer executable, which will be named something like:

  • MarvinsAIRARefactored-Setup-2.x.xxx.xxxx.exe

Save the file somewhere easy to find, such as your Desktop or Downloads folder.

If you already have an older MAIRA Refactored version installed, you can simply install the newer version over the top. Your settings will be preserved.

Step 2 — Run the Installer

Double-click the downloaded MarvinsAIRARefactored-Setup-2.x.xxx.xxxx.exe file to start the installer. Windows may show a SmartScreen warning because the file was downloaded from the internet.

I ain’t paying $300 a year to Microsoft for a certificate to remove this whine screen. If you see this, choose More info and then Run anyway to continue. Or, if you don’t trust me, then don’t. The source code for MAIRA is public and can be reviewed by anyone.

The installer will walk you through a short setup wizard where you can choose the installation location (the default is usually fine).

Tip: If you have SimHub running, please close it before installing MAIRA Refactored.

Step 3 — Finish and Launch MAIRA

When the installer finishes, it will add shortcuts for MAIRA to your Start Menu, and you can also optionally create a Desktop shortcut.

To start MAIRA, use either:

  • The Desktop shortcut, if you created one.
  • The Start Menu entry under your installed programs.

On first launch, MAIRA will create its configuration files and data folders automatically. This may take a moment the very first time.

Keeping MAIRA Up to Date

New versions of MAIRA Refactored are released on the same GitHub Releases page:

https://github.com/mherbold/MarvinsAIRARefactored/releases/latest

Whenever a new version is available, download the newer installer and run it. You do not need to uninstall the previous version first.

By default, the app will also let you know if a newer version is available, and you can choose to download and install it automatically.
