Getting Started

Basic first-time setup for MAIRA Refactored (2.0).

Overview

This page walks through the basic setup steps after installing MAIRA Refactored (2.0). The overall concepts are similar to the original MAIRA app, but many settings now live in separate, focused windows with a refreshed UI.

Before You Start

For the smoothest experience, it’s best to have iRacing running in a test session while you configure MAIRA. That way, telemetry is live and MAIRA can immediately detect your car, track, and wheel.

  • Launch iRacing and MAIRA.
  • Join a test session with any car/track combination.
  • Wait for the test session to load and for MAIRA to recognize that iRacing is running.
Important note for Logitech wheel users: You must also disable the iRacing simulator’s support for TrueForce, prior to running the iRacing simulator. To do this, edit the app.ini file which is located in the documents iRacing folder, and find and replace loadTrueForceAPI=1 with loadTrueForceAPI=0 and save.

Launching MAIRA

Start MAIRA using the Desktop or Start Menu shortcut. When it opens, it will automatically try to connect to iRacing and your configured hardware.

If MAIRA doesn’t see iRacing yet, that’s okay — the app will connect automatically as soon as iRacing is in a session and telemetry is available.

Launching iRacing

When you launch iRacing, you need to go into the settings and disable force feedback in iRacing.

MAIRA will detect whether force feedback is enabled in iRacing, and will disable itself if it is.

Basic Configuration Steps

  1. Select your racing wheel device.
  2. Configure the overall strength settings by setting the wheel force according to your wheelbase and setting the max force to about 30 Nm, which is a good starting spot.
  3. Map the + and - buttons for the strength setting to a knob or buttons on your wheel.

You’re Ready to Drive

Once you have done the above, you're ready to start driving! Turn up or down the overall strength until you are satisfied with the weight of the wheel. Then once you are ready to get into some of the more advanced features and fine tuning, check out the rest of this documentation.
